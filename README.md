## Project Overview
This project is an extension of the training work using the Google Play Store dataset. The objective is to perform additional analysis and create interactive visualizations based on specific conditions. All tasks are implemented using the same dataset as required.

## Dataset
The dataset used in this project:
1) Google Play Store Apps dataset
2) Google Play Store Reviews dataset

## Tools Used
•	Python 
•	Pandas 
•	Plotly Express 
•	HTML/CSS 

## Tasks Implemented
### Task 1: Category Analysis
A grouped bar chart comparing average rating and total reviews for top categories based on installs. Filters include rating, size, and update month.

### Task 2: Installs Visualization
A choropleth map showing installs distribution for top categories with specific filtering conditions.

### Task 3: Free vs Paid Comparison
A comparison of average installs for free and paid apps within top categories using filtered data.

### Task 4: Time Series Analysis
A line chart showing installs trend over time by category, with highlighted periods of high growth.

### Task 5: Bubble Chart Analysis
A bubble chart showing the relationship between app size and rating, with installs represented as bubble size and filtered categories.

### Task 6: Stacked Area Chart
A stacked area chart showing cumulative installs over time for selected categories with growth highlights.

## Features
•	Interactive charts using Plotly 
•	Multiple filters applied as per task requirements 
•	Time-based visibility for each visualization 
•	Clean and structured dashboard layout 

## Dashboard
All visualizations are combined into a single HTML dashboard. Each graph is displayed only within its specified time window.
